第一步：下载 data.json
点击下方按钮，将当前 0 条优惠券数据导出为 data.json 文件，保存到你的电脑。
第二步：上传到你的托管平台
将下载的 data.json 上传到你的网站根目录，与 index.html 放在同一个文件夹。

常用平台操作：
· GitHub Pages → 进入仓库，上传 data.json 提交
· 阿里云 OSS → 进入 Bucket，上传并覆盖同名文件
· 腾讯云 COS → 进入存储桶，上传覆盖
· Vercel → 把 data.json 放进项目文件夹重新部署
第三步：验证生效
刷新前台页面，数据即时生效。如果看不到更新，按 Ctrl+Shift+R（Mac 用 Cmd+Shift+R）强制刷新清除缓存。
📁 正确的文件结构
你的网站根目录/
├── data.json      ← 每次发布后更新这个
├── index.html     ← 前台页面（用户看到的）
└── admin.html     ← 后台管理（你用的）
